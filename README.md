Blinkit Sales Analysis Dashboard | Guided Power BI Project
Overview

This is a guided Power BI project focused on building an interactive sales dashboard using Blinkit's grocery sales dataset. The project covers the complete dashboard development process, including data cleaning, DAX calculations, field parameters, interactive visualisations and dashboard formatting.

Project Objectives
Analyse overall sales performance
Compare product and outlet performance
Track sales trends over time
Build an interactive dashboard using Power BI best practices
Dashboard Highlights
KPIs
Total Sales
Average Sales
Number of Items
Average Rating
Visualisations
Donut Chart – Sales by Fat Content
Bar Chart – Sales by Item Type
Stacked Column Chart – Fat Content by Outlet
Line Chart – Sales by Outlet Establishment Year
Donut Chart – Sales by Outlet Size
Funnel Chart – Sales by Outlet Location
Matrix – Outlet Type Performance
Data Preparation
Imported Excel data into Power BI
Performed data cleaning in Power Query
Standardised inconsistent values (LF → Low Fat, Reg → Regular)
Validated data quality using Column Quality
Applied transformations using Applied Steps
DAX Measures

Created DAX measures for:

Total Sales
Average Sales
Number of Items
Average Rating

Interactive Features
Field Parameters for dynamic KPI selection
Slicers for Outlet Location, Outlet Size and Item Type
Cross-filtering using Edit Interactions
Interactive dashboard navigation
Tools & Technologies
Power BI Desktop
Power Query
DAX
Microsoft Excel
